react + antd-mobile项目的搭建

发表于 2019-03-08 | 分类于 React | 浏览 次

react的学习

有段时间，想要换份工作，联系了几家，居然都要技术栈是 react, 虽然对vue 已经单独完成过几个项目，已经基本掌握vue的搭建构建以及部署的流程，还在继续逐步研究源码。
对react 一直都是虽有学习，但是未涉及实战项目，心有不甘，既然嫌弃我没做过react项目，又来个新项目的机会，决定挑战一下，遂自己从零开始搭建项目，来个实战开发。

react 搭建 （中文官网地址：）

  • React 是一个用于构建用户界面的 JavaScript 库。

  • Create React App

    • 是一个用于学习 React 的舒适环境，也是用 React 创建新的单页应用的最佳方式。
    • 它会配置你的开发环境，以便使你能够使用最新的 JavaScript 特性，提供良好的开发体验，并为生产环境优化你的应用程序。你需要在你的机器上安装 Node >= 6 和 npm >= 5.2。要创建项目

  • 浏览器打开 localhost:3000 就可以看到效果了

引入 antd-mobile

  • 按需加载

    • 引入 react-app-rewired 并修改 package.json 里的启动配置。。由于新的 react-app-rewired@2.x 版本的关系，你需要还需要安装 customize-cra。
    • 然后修改 package.json
    • 然后在项目根目录创建一个 config-overrides.js 用于修改默认配置。
    • 使用 babel-plugin-import, babel-plugin-import 是一个用于按需加载组件代码和样式的 babel 插件（原理），现在我们尝试安装它并修改 config-overrides.js 文件。

    • 更改引用方式

antd-mobile 定制主题

antd-mobile 的样式使用了 Less 作为开发语言
摘自可点击查看此博客

LESS
  • 第一步：安装less less-loader
  • 第二步：在 package.json 文件中添加一个 theme 字段，里面将包含所有我们想要修改的主题样式
  • 第三步：修改config-overrides.js文件，覆盖webpack配置
SCSS
  • 第一步：安装sass-loader
  • 第二步：修改config-overrides.js文件，覆盖webpack配置

然后你就可以使用 npm start 来查看页面的结果了
