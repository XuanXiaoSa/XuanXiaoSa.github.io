javascript闭包

闭包是指那些能够访问独立(自由)变量的函数 (变量在本地使用，但定义在一个封闭的作用域中)。换句话说，这些函数可以“记忆”它被创建时候的环境。(难以理解的说法)

  • 个人理解

    闭包就是 可以使函数外部访问函数内部变量的一个桥梁函数。

  • 一个函数（a）本本身有自己的作用域，函数(a)外部是无法访问函数(a)内部定义的变量的，但是可以在函数(a)的内部定义一个函数(b) return 出去，在函数（a）外面定义一个变量c, 将函数a 调用之后 （ a() ）赋值给 c 再调用c 就相当于在外部调用函数（b）就可以拿到函数（a)里面的声明的变量了。
