http 以及 缓存

  • 缓存

    • 强缓存：不会发起请求。
  • 协商缓存：会发起一次请求。

这两组搭档都是成对出现的，即第一次请求的响应头带上某个字段（Last-Modified或者Etag），则后续请求则会带上对应的请求字段（If-Modified-Since或者If-None-Match），若响应头没有Last-Modified或者Etag字段，则请求头也不会有对应的字段。
