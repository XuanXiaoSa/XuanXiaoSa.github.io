博客搭建过程

前段时间一直忙于项目，整天加班，鉴于最近闲下来了，就开始构思搭建自己的博客了，目的主要有以下几点：

  • 完善自己的信息
  • 将自己的笔记整理到上面，便于以后管理和更新
  • 自我升华（装 13）

项目构思

首先要想想你的博客项目想要什么样子的，不然怎么开始做呢，网上有很多，暂时没找到合乎自己审美（kou wei，又再装13），所以就想着自己设计搭建一个吧。

  • 不错的博客 markdown主题 - next
  • 博客需要的功能或者需求（主要看看别人的都有啥，比较好的借鉴一下下）
    • 导航菜单（个人主页、随笔、其他）
    • 时间分类（以时间，年月日分类）
    • 文章分类（文章种类、js/css/vue/小程序/RN/计算机基础等等）
    • 联系社交（github、其他技术网站账号）
